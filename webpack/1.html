• 1.安装：

    $ npm install webpack -g

  • 2.编译

    $ webpack ./entry.js bundle.js

  • 3.引用其它文件

    require("./content.js")

  • 4.处理 css文件
    webpack can only handle JavaScript natively

    npm install css-loader style-loader

    - require("!style!css!./style.css"); + require("./style.css"); webpack ./entry.js bundle.js --module-bind "css=style!css"
  • 5.配置文件

    add webpack.config.js

    module.exports = { entry: "./entry.js", output: { path: __dirname, filename: "bundle.js" }, module: { loaders: [ { test: /\.css$/, loader: "style!css" } ] } }; 写了配置文件后，只需要运行 webpack 显示编译进度 webpack --progress --colors
  • 6.代码修改后自动编译 webpack --watch
  • 7.部署服务器
