前端开发知识点

基础、核心知识（必学）

  • html、html5
  • css css3
  • js (常用js 库：jquery、zepto，必须会用)

相关网站

  • github(查找项目或源码)
  • http://www.w3school.com.cn（入门、基础知识、知识手册）
  • developer.mozilla.org（知识手册）
  • sass教程
  • udacity(英文视频教程,我之前看的就是这个)
  • Bootstrap公共cdn（上面的星星来源于github，星星越多，表示这个插件越多人喜欢）

预处理器

用于生成css和js代码的语言。常用的有css预处理器有sass、less、stylus，常用的js预处理器有coffeescript、typescript。 css预处理器一般要求会一个，现在sass比较流行。js预处理器,有些要求，有些不要求，属于加分项，可以不学，es6是最新的js语法，也是加分项，可以不学
